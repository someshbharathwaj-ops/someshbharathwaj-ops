## About

I'm a computer science student pursuing a dual-degree in CSE and Data Science & AI. My work sits at the intersection of **retrieval systems**, **applied ML**, and **algorithmic thinking**. I care about building things that actually work — not demos, but systems with clean architecture and measurable behavior.

Currently deepening expertise in RAG system design, evaluation pipelines, and the fundamentals that make ML reliable in production.
